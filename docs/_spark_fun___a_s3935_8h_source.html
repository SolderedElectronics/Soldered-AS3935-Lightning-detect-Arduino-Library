160 // REG0x03, bits [3:0], manufacturer default: 0.
161 // When there is an event that exceeds the watchdog threshold, the register is written
162 // with the type of event. This consists of two messages: INT_D (disturber detected) and
163 // INT_L (Lightning detected). A third interrupt INT_NH (noise level too HIGH)
164 // indicates that the noise level has been exceeded and will persist until the
165 // noise has ended. Events are active HIGH. There is a one second window of time to
166 // read the interrupt register after lightning is detected, and 1.5 after
167 // disturber.
